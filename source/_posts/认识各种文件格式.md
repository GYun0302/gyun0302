# xml文件
参考链接：https://developer.mozilla.org/zh-CN/docs/Web/XML/Guides/XML_introduction

简介：xml (Extensible Markup Language)是一种可扩展标记语言。不依赖于软硬件平台，是可以在不同系统之间交换数据的通用标准。
格式：
    类型：纯文本
    结构：标签、属性、文本内容。
示例：
```
<note>
  <to>用户</to>
  <from>此博客</from>
  <heading>提醒</heading>
  <body>这是一个XML文件示例。</body>
</note>
```
# npz文件
简介：npz (NumPy Zip Archive)是NumPy 的压缩归档文件。用来保存多个数组，可以将多个处理好的矩阵或者张量打包到一个文件中，方便读取和移动。比起文本格式（如 CSV）小，支持压缩且读写速度快。
格式“
    类型：二进制
    结构：本质上是一个ZIP压缩包。里面包含了一个或多个.npy文件，每个.npy对应一个数组，以及一个用于索引的.pkl 文件。
示例：
```
import numpy as np
data = np.load('dataset.npz')
x_train = data['x']  # 获取名为 'x' 的数组
y_train = data['y']
```
# obj文件
简介：通用的3D模型文件格式（最初由 Wavefront Technologies 开发）。结构很简单，只存储几何信息（顶点、面、纹理坐标、法线），不存储动画、骨骼或复杂材质。
格式：
    类型：纯文本
    结构：每一行代表一种几何元素。
        v：顶点 (Vertex) 坐标 (x, y, z)
        vt：纹理坐标 (Texture coordinates)
        vn：法线 (Normals)
        f：面 (Face)，定义由哪几个顶点组成一个面
示例：
```
# 这是一个简单的立方体或者三角形片段
v 1.0 2.0 3.0   # 第1个顶点
v 4.0 5.0 6.0   # 第2个顶点
v 7.0 8.0 9.0   # 第3个顶点
f 1 2 3         # 一个面，由第1、2、3号顶点组成
```
